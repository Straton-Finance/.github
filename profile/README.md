### About

Straton is a regulated platform for tokenizing real-world assets (RWA) on the Base blockchain. We combine institutional-grade smart contracts with a seamless user experience to bridge traditional finance and DeFi.

### Key Features

- **ERC-3643 Compliant** — Security token standard with on-chain identity verification and modular compliance
- **Institutional Vault** — Deposit stablecoins (USDC/USDT), receive receipt tokens, earn yield
- **KYC/AML Pipeline** — Automated identity verification with on-chain whitelisting
- **Sanctions Screening** — Chainalysis oracle integration for real-time OFAC compliance
- **Multisig Governance** — Safe-based admin operations for mint, yield, and withdrawals

### Tech Stack

`Solidity` · `Foundry` · `Next.js` · `TypeScript` · `Supabase` · `wagmi` · `viem` · `Base L2`
